# License-Plate-Recognition

Este proyecto implementa un sistema automático de segmentación y reconocimiento de caracteres en placas de matrícula a partir de imágenes.

El sistema recibe como entrada una imagen de una matrícula y:

- Segmenta automáticamente cada carácter.
- Obtiene su bounding box y centroide.
- Reconoce cada carácter mediante correlación con plantillas.
- Muestra la imagen original con los contornos detectados y la matrícula reconocida como título.
